CineManager is a mobile cinema ticket booking application designed to provide a simple, intuitive, and accessible experience without forcing users to create an account. Users can browse movies currently showing, explore available cinemas, check screenings, and reserve tickets in just a few steps. On the technical side, the app is built with React Native and Expo, and uses SQLite for local storage of reservations and tickets, enabling offline-first usage without any dependency on a remote server. The project architecture focuses on clear separation of concerns, reusable components, and maintainable code, following modern mobile development best practices.
